# Testing with pytest

pytest is the de facto standard for Python testing. Its fixture system and parametrization make writing comprehensive tests easier than any other Python test framework.

## Basic Tests

```python
# test_math.py
def test_addition():
    assert 1 + 1 == 2

def test_string():
    assert "hello".upper() == "HELLO"
```

## Fixtures

```python
import pytest

@pytest.fixture
def user_data():
    return {"name": "Alice", "age": 30}

@pytest.fixture(scope="session")
def db_connection():
    """Created once per test session."""
    conn = create_database()
    yield conn
    conn.close()

def test_user_greeting(user_data):
    assert f"Hello, {user_data['name']}" == "Hello, Alice"
```

## Key Takeaways

- Fixtures replace setUp/tearDown with explicit dependency injection
- Parametrization eliminates duplicated test code
- Mock external APIs, not your own code
- Aim for fast, isolated, deterministic tests
