# Package Management with uv

The Python packaging ecosystem has evolved significantly. `uv` is the modern standard — 10-100x faster than pip, with built-in virtual environment management.

## Project Setup

```toml
# pyproject.toml
[project]
name = "my-project"
version = "0.1.0"
requires-python = ">=3.11"
dependencies = [
    "requests>=2.31",
    "pydantic>=2.0",
]

[build-system]
requires = ["setuptools>=68"]
build-backend = "setuptools.build_meta"
```

## uv Commands

```bash
# Create virtual env and install deps
uv venv
uv sync

# Add a dependency
uv add requests

# Run a command in the virtual env
uv run python main.py
```

## Key Takeaways

- Use `pyproject.toml` instead of `setup.py`
- `uv` is 10-100x faster than pip — switch if you haven't
- Lock files ensure reproducible builds
- Always use virtual environments — never install globally
