#これら9つのオープンソースAIおよびMCPプロジェクトで開発者の生産性を加速

GitHub CopilotとVS Codeは、Microsoft Open Source Program Office （ OSPO ）とともに、9つのオープンソース** Model Context Protocol （ MCP ）**プロジェクトを後援しています。  これらのプロジェクトは、AIネイティブのワークフロー、エージェントツーリング、開発者がコードベース、ブラウザ、現実世界のアプリケーションと対話するための新しい方法を解放します。

MCPを使用すると、開発者は人工知能と自律エージェントを推進して、幅広いエコシステムにわたってコードを理解、操作、さらには実行することができます。  ブラウザの拡張機能からゲームエンジンの統合まで、新しいツールはAIファーストの生産性を日常の開発者エクスペリエンスにもたらします。

##メジャーテーマ

9つのプロジェクトは、** 3つの繰り返し発生する原則**に基づいています。
1. **フレームワークとプラットフォームの統合** – MCPを一般的なエコシステムに接続します。
2. **開発者の経験とAI強化コーディング** – LLMがインテリジェントなIDEアシスタントとして機能します。
3. **自動化、テスト、オーケストレーション** – MCPを生産レベルの信頼性に引き上げます。

## 3.自動化、テスト、オーケストレーション

これらのプロジェクトは、MCPが本番環境で確実に拡張できるようにし、堅牢なテスト、デバッグ、およびワークフローツールを提供します。

|プロジェクト|目的|
|--------|--------|
|' n 8 n - mcp '| n 8 nのワークフローの自動化を促進する超最適化されたプラットフォーム。 ノードオーケストレーションを合理化し、AIモデルを統合して、ユーザーがn 8 nノードをよりよく理解し、操作できるようにします。|
|' inspector '| MCPサーバーのデバッグツールキット。 プロトコルハンドシェイク、ツール登録、リソース定義、プロンプト、およびOAuthフローを検査し、回帰をキャッチするための組み込みのLLMプレイグラウンドシミュレーションを使用します。|
