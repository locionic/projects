> **Tóm tắt nhanh**
> - pyenv cho phép cài nhiều Python versions trên cùng machine và switch linh hoạt
> - Installation gồm: install dependencies → download pyenv → configure shell → verify
> - Dùng `pyenv versions` để xem tất cả installed versions, `pyenv global X.Y.Z` để set default

## Cài đặt pyenv trên Ubuntu

### 1. Install Dependencies

```bash
apt update -y
apt install -y make build-essential libssl-dev zlib1g-dev libbz2-dev \
  libreadline-dev libsqlite3-dev wget curl llvm libncurses5-dev \
  libncursesw5-dev xz-utils tk-dev libffi-dev liblzma-dev \
  python-openssl git
```

Tất cả dependencies là required để compile Python từ source. Thiếu packages sẽ gây lỗi khi install Python versions.

### 2. Download và Install pyenv

```bash
curl https://pyenv.run | bash
```

Script này clone pyenv repo và setup shell configuration.

### 3. Configure Shell

```bash
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
echo -e 'if command -v pyenv 1>/dev/null 2>&1; then\n  eval "$(pyenv init -)"\nfi' >> ~/.bashrc
exec "$SHELL"
```

Sau khi chạy, shell sẽ recognize `pyenv` command.

### 4. Verify Installation

```bash
pyenv install --list     # Xem tất cả Python versions available
pyenv install 3.12.0     # Cài một version cụ thể
pyenv versions           # Xem versions đã cài
pyenv global 3.12.0      # Set version mặc định
```

Output của `pyenv versions` sẽ hiển thị dấu asterisk (*) bên cạnh global version.

### Useful Commands

```bash
pyenv help              # Xem tất cả commands có sẵn
pyenv local 3.11.0      # Set local version cho directory hiện tại
pyenv shell 3.10.0      # Override version cho current shell session
```

## FAQ

### pyenv vs pyenv-virtualenv?

`pyenv-virtualenv` là plugin cho pyenv, tự động tạo virtualenv khi chạy `pyenv local` hoặc `pyenv global`. Nếu dùng poetry hoặc pipenv riêng, có thể không cần.

### Làm sao uninstall Python version?

```bash
pyenv uninstall 3.11.0
```

### pyenv install chậm?

Compile Python từ source tốn thời gian. Trên Ubuntu có thể cài pre-built binaries qua `pyenv install X.Y.Z` sau khi install `xorg-dev` và `libsqlite3-dev`.

### Multiple Python versions cùng lúc?

pyenv switch Python version theo scope: global (machine-wide), local (directory), shell (current session). Priority: shell > local > global.
