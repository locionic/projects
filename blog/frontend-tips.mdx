> **Tóm tắt nhanh**
> - Thêm `border: 1px solid transparent` trước để hover border không làm layout nhảy
> - `box-sizing: border-box` hoặc `outline` là hai cách implement phổ biến
> - Padding compensation cũng works nhưng cần tính toán lại giá trị

## Ngăn layout nhảy khi thêm border on hover

Khi hover vào một element mà border xuất hiện đột ngột, nội dung xung quanh sẽ bị đẩy ra — looks bad. Có 3 cách giải quyết.

### Cách 1: Outline

```css
.your-element {
  border: 1px solid transparent;
  outline: 1px solid transparent;
  transition: all 0.3s ease;
}

.your-element:hover {
  border-color: #333;
  outline-color: #333;
}
```

Outline không ảnh hưởng đến kích thước element — border nằm ngoài box model, outline vẽ phía ngoài border. Layout không shift.

### Cách 2: Box-sizing

```css
.your-element {
  box-sizing: border-box;
  border: 1px solid transparent;
  transition: all 0.3s ease;
}

.your-element:hover {
  border-color: #333;
}
```

Với `box-sizing: border-box`, padding và border được tính vào tổng width/height. Transparent border đã chiếm không gian sẵn nên hover không thay đổi kích thước thực.

### Cách 3: Padding Compensation

```css
.your-element {
  border: 1px solid transparent;
  padding: 10px;
  transition: all 0.3s ease;
}

.your-element:hover {
  border-color: #333;
}
```

Cách này đặt padding trước để border chiếm không gian tương đương. Trade-off: phải tính toán padding compensation cẩn thận, và nếu border width thay đổi thì padding cũng phải adjust.

## FAQ

### Cách nào là best practice?

`outline` là an toàn nhất vì không affect box model với border. Nhưng `box-sizing: border-box` phổ biến hơn trong các design system vì nó solve nhiều issues khác nữa — dùng khi project đã set `box-sizing: border-box` globally.

### Transition nên đặt ở đâu?

Đặt transition trên element không có `:hover` — có nghĩa là transition khi hover vào VÀ khi hover ra đều smooth. Giá trị `0.3s` là sweet spot cho UI, `transition: all` đôi khi quá nhiều, nên specify rõ `transition: border-color 0.3s ease, outline-color 0.3s ease`.
