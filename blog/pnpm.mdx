# Installation

Want to try pnpm on your machine? Here's the fastest way to get it running.

## What is pnpm and Why Bother?

pnpm is a package manager that stores modules in a shared location on your disk, then symlinks them into individual projects. The result? Faster installs and way less disk space eaten up by node_modules.

If you're still on npm or yarn, give pnpm a shot. You'll notice the difference.

## Set It Up in 30 Seconds

Fire up your terminal and run:

```
npm install --global corepack@latest
corepack enable pnpm
```

That's it. Corepack comes with Node.js (v16 and up), so you probably already have it.

## Quick Notes

- Run `pnpm --version` to confirm it installed correctly.
- When you `pnpm install` in a project, it reads the `packageManager` field in package.json automatically.
- If a project uses a specific pnpm version, Corepack ensures everyone uses the same one.

## Common Commands

| Command | What it does |
|---------|--------------|
| `pnpm install` | Install all dependencies |
| `pnpm add <package>` | Add a new dependency |
| `pnpm update` | Update packages |
| `pnpm remove <package>` | Remove a dependency |

## Key Takeaways

- pnpm is faster and uses less disk space than npm or yarn
- Corepack is the easiest way to install and manage pnpm versions
- Run `corepack enable pnpm` to get started in seconds
- It reads `packageManager` from package.json automatically

## FAQ

**How is pnpm different from npm?**

pnpm uses a content-addressable store, so the same package version is stored only once on your disk. npm copies packages into each project's node_modules, which takes more space.

**Do I need to uninstall npm first?**

No. You can have pnpm and npm installed at the same time. They don't conflict.

**Can I use pnpm with existing projects?**

Yes. Just run `pnpm install` in any project with a package.json and it will work.

**What if a project has a specific pnpm version requirement?**

Corepack handles this automatically. It reads the `packageManager` field and uses the correct version.

**Does pnpm work with monorepos?**

Absolutely. pnpm has excellent monorepo support built in.
