# Java OOP (Object-Oriented Programming)

## Vietnamese

- 4 tính chất OOP1321:
  - 1. Encapsulation (đóng gói) - che giấu dữ liệu, chỉ cho phép truy cập qua getters/setters.
  - 2. Inheritance (kế thừa) - tái sử dụng code, class con kế thừa thuốc tính/phương thức từ class cha.
  - Polymorphism (đa hình) - cùng một phương thức nhưng hành vi khác nhau (overloading, overriding)

- Các khái niệm quan trọng:
 - Interface vs Abstract class
 - Overloading vs Overriding
 - Access modifiers (public, private, protected, default)
 - Exception handling (try-catch-finally, throws)
 - Collections framework (list, set, map)

## English

- 4 OOP principles:
  - 1. Encapsulation - hiding data, accessing via getter/setter.
  - 2. Inheritance - reusing code, child class inherits from parent class.
  - 3. Polymorpphism - same method, different behavior (overloading, overriding).
  - 4. Abstraction - hiding implementation, exposing only interface/abstract class.
- Impportant concepts:
  - Interface vs Abstract class
  - Overloading vs Overriding
  - Access modifiers (public, private, protected default).
  - Exception handling (try-catch-finally, throws)
  - Collections framework (List, Set, Map)

- Key concepts:
  - Modern test framework by Microsoft, supports Java, Python, C#, Node.js.
