# Vim Search and Replace Tips

## English Version

### 1. Replace function calls but keep difinition
- Use `:g!` to exclude the function definition line while replacing
- Example:
```vim
:%g!/^\s*def test_not_none/ s/\<test_not_none(/test_not_alone(self.report, /
```
- `:%` -> while file.
- `g!` -> run substitution on lines not matching the given pattern.
- `\<` -> ensures whole word match.
### 2. Meaning of `g` in Vim
- In `:s/foo/bar/g`:
  - `g` = **global in the line** (replace all matches on that line)
  - Memory trick: **"greedy"**
### 3. Search for `f` and `F`
- Case insentitive search:
  - `:set ignorecase` or `/f\c`.
- Regex search:
  - `/[fF]`.
### 4. Search `"abcd"` in Current Line
- Restrict search to current line:
```vim
:.,.s/abcd/&/
```
- Regex for current line:
```vim
/\%l.*abcd
```
- Visual selection + substitude:
  1. Press `V` to select line.
  2. Run `:s/abcd/&/`.
