# Main categories of webui keywords & key functions
| Category | Keywords | What they do |
| -------- | -------- | ------------ |
| Navigation & Browser Control | `openBrowser()`, `closeBrowser()`, `navigateToUrl()`, `refresh()`, `back()`, `forward()` | Open/close browser, navigate pages |
| Waiting/Synchronization | `waitForElementVisible()`, `waitForElementClickable()`, `waitForElementPresent()`, `waitForPageLoad()` | Wait until elements or page states satisfy conditions |
| Click & interaction | `click()`, `doubleClick()`, `rightClick()`, `hover()`, `focus()` | Interact with elements (buttons, links, etc.) |
| Selection/Dropdown | `selectOptionByLabel()`, `selectOptionByValue()`, `selectOptionByIndex`, `deselectOptionByValue()`, `selectAllOption()` | Work with `<select>` elements and dropdown |
| Check/toggle | `check()`, `uncheck()` | Checkboxes, toggles |
| Alerts/popups | `acceptAlert()`, `dismissAlert()`, `getAlertText()` | Handle browser alert dialogs |
| Frame/window | `switchToFrame()`, `switchToWindowIndex()`, `switchToDefaultContent()` | Switch context between frames or windows |
| Verification | `verifyElementPresent()`, `verifyElementVisible()`, `verifyTextPresent()`, `verifyMatch()`, `verifyEqual()` | Validate UI state or values |
| Retrieving properties | `getText()`, `getAttribute()`, `getCssValue()`, `getElementWidth()`, `getElementHeight()` | Read values or style attributes from UI |
| Find elements | `findWebElements()`, `findWebElement()` | return WebElement for further processing |
| Javascript execution | `executeJavaScript()` | Run JS snippets in browser context |
| Visual/misc | `scrollToElement()`, `takeScreenshot()`, `focus()`, `highlight()` | Misc UI helpers |
| Authentication | `authenticate()` | Handle http basic auth (username/password) |
| Window/viewport info | `getWindowTitle()`, `getViewportWidth()`, `getViewportHeight()` | Get infor about browser window, viewport |
