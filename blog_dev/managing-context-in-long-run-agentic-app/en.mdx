# Managing Context in Long-Run Agentic Applications

In complex, long-running agentic systems, maintaining alignment and coherent reasoning between agents requires careful design. This article explores the challenges and presents a range of complementary techniques to balance continuity and creativity.

## The Challenge of Long-run Coherence

Language model APIs are stateless, requiring the caller to provide the complete message history with each request to maintain continuity. Agent frameworks solve the state management problem for users by accumulating message history between API calls. However, even approaching an agent's context window limit can degrade the quality of responses.

## Complex Security Investigations

Complex security investigations can span hundreds of inference requests and generate megabytes of output, requiring special handling. Multi-agent applications, like ours, add further complexities. For each agent to optimally execute its role, it requires a tailored view of the investigation state. Each view must be carefully balanced to avoid overwhelming agents or stifling creativity.

## Our Solution: Three Complementary Context Channels

Our solution uses three complementary context channels:

* **Director's Journal**: The Director's structured working memory, capturing decisions, observations, hypotheses, and open questions in a structured format.
* **Critic's Review**: An annotated findings report with credibility scores, summarizing the Experts' findings and methodology.
* **Critic's Timeline**: A consolidated chronological findings report with credibility scores, providing a high-level overview of the investigation.

## How Our Agents Consume and Produce Different Context Sources

Each agent receives the current content of the Director's Journal in their prompt, presented as a chronology. The Critic receives a summary report and uses a suite of tools to examine the cited evidence.

## The Director's Journal

The Director's Journal captures decisions, observations, hypotheses, and open questions in a structured format. It serves as the Director's working memory, allowing it to lead the investigation towards a conclusion, observe and measure progress, identify dead-ends, and make course corrections.

## The Critic's Review Tools

The Critic's role is to assess the Experts' work, checking that reported findings are supported by evidence and that interpretations are sound. To do this accurately, it needs to be able to inspect not only each Expert's claims and the cited evidence but the methodology.

## The Critic's Rubric

We use a five-level credibility scale to score findings:

* **Trustworthy**: Supported by multiple sources with no contradictory indicators
* **Highly-plausible**: Corroborated by a single source
* **Plausible**: Mixed evidence support
* **Speculative**: Poor evidence support
* **Misguided**: No evidence provided or misinterpreted

## Mitigations for the Critic's Review

We approach the problem of the Critic's Review providing a false sense of assurance from several directions with a range of mitigations:

* Using a stronger model for the Critic
* Formulating the Critic's instructions to reduce the likelihood of hallucination
* Improving the Critic's capacity to interpret nuances in the evidence with a stronger model

By using these complementary techniques, we can balance continuity and creativity in complex, long-running agentic systems, ensuring that our agents work productively over long time spans.
