# Harnessing Interrogatory LLMs for Context Creation and Knowledge Extraction

When working with Large Language Models (LLMs) on complex tasks, providing extensive context is often essential. This context can encompass various aspects, such as user interface descriptions, implementation guidelines, and information on external systems. While humans can write this context, an alternative approach is to leverage an LLM to generate it through an interrogative process.

## Interrogatory LLMs: A Novel Approach

This approach involves prompting an LLM to ask questions to gather necessary information. The LLM can then create a context report based on the collected data, which can be used in subsequent sessions, potentially with different models. A crucial aspect of this technique is to ensure the LLM asks only one question at a time, as suggested by Harper Reed in his blog post.

## Applications of Interrogatory LLMs

1. **Context Creation**: Interrogatory LLMs can be used to generate context for complex tasks, such as designing new features. By asking relevant questions, the LLM can gather necessary information and create a comprehensive context report.
2. **Document Review**: Interrogatory LLMs can also be employed to review documents, such as software specifications, by interviewing human experts. This approach can be more effective than traditional review methods, especially for poorly written documents.
3. **Knowledge Extraction**: Interrogatory LLMs can facilitate knowledge extraction from individuals who struggle with writing. By asking questions and generating text based on the responses, LLMs can help capture information that might otherwise remain unrecorded.

## Benefits and Implications

The use of interrogatory LLMs can be beneficial in various scenarios:

* **Improved productivity**: By automating context creation and document review, interrogatory LLMs can save time and increase productivity.
* **Enhanced knowledge extraction**: Interrogatory LLMs can help capture information from individuals who struggle with writing, reducing the risk of lost knowledge.
* **Broader applications**: This technique can be applied in various domains, from software development to research and documentation.

However, it is essential to acknowledge the potential drawbacks, such as the "tang of AI-writing" that may be present in the generated text. Nevertheless, the benefits of using interrogatory LLMs in context creation and knowledge extraction make them a valuable tool in the AI landscape.
