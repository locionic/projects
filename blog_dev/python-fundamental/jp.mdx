# 毎日使うPythonの基礎（これだけで十分）

Pythonでプログラムを作るということは、結局「データを扱う（変数）」と「次に何をするか決める（制御フロー）」の組み合わせに過ぎません。ここでは絶対に外せない基本ブロックを紹介します。

<Callout type="idea" title="Pythonの黄金律">
インデント（字下げ）がすべてです。波括弧 `{}` の代わりに空白を使います。インデントがずれていると、コードは動きません。
</Callout>

---

## 4つの基本ブロック

<Stepper>
  <Step title="1. 変数と型（Types）">
    型を気にする必要はありませんが、無視してはいけません。`"5" + 5` のようなミスが初心者の最大の敵です。
    - `int` (整数), `float` (小数)
    - `str` (文字列), `bool` (True/False)
  </Step>

  <Step title="2. 入力と変換 (Input)">
    `input()` は**必ず文字列を返します**。計算に使いたいなら `int()` などですぐに変換しましょう。
    ```python
    age = int(input("Age: "))
    print(f"来年は {age + 1} 歳ですね。")
    ```
  </Step>

  <Step title="3. 条件分岐 (if/elif/else)">
    条件の比較には `==` (等しい), `<`, `>` などを使い、インデントでブロックを作ります。
    ```python
    if score >= 90:
        print("Grade: A")
    elif score >= 80:
        print("Grade: B")
    ```
  </Step>

  <Step title="4. ループ (for & while)">
    同じことを繰り返すための構文です。
    - 回数が決まっているなら `for`
    - 条件が変わるまで繰り返すなら `while`
  </Step>
</Stepper>

## データ構造と再利用

<Callout type="info" title="リスト (Lists)">
複数のデータをまとめる一番簡単な方法です。インデックスは `0` から始まります。
```python
nums = [10, 20, 30]
print(nums[0])  # 10
```
</Callout>

<Callout type="info" title="関数 (Functions)">
コピペをやめましょう！同じ処理は `def` で関数にまとめると、バグ修正が一度で済みます。
```python
def greet(name):
    return f"Hi, {name}!"
```
</Callout>

<Callout type="warning" title="エラーと友達になる">
`SyntaxError` や `ValueError` が出ても焦らないでください。エラーメッセージ（Traceback）を読むことこそが、Python習得の最短ルートです。
</Callout>
