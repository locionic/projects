# Viết Code Python Đầu Tiên Ngay Hôm Nay

Đừng chỉ xem các tutorial dài lê thê nữa. Học lập trình là phải tự gõ code và chạy thử trên máy của mình. Dưới đây là hướng dẫn siêu tốc để bạn bắt đầu.

<Callout type="info" title="Bước 0: Cài đặt Python">
Tải Python từ **python.org**. Nếu dùng Windows, **BẮT BUỘC** tick vào ô **"Add Python to PATH"** khi cài đặt. Mở terminal và gõ `python --version` để kiểm tra.
</Callout>

---

## Các Khái Niệm Cốt Lõi

<Stepper>
  <Step title="1. Biến & Kiểu dữ liệu">
    Không cần khai báo kiểu phức tạp, chỉ cần đặt tên và gán giá trị!
    ```python
    name = "Ava"       # Chuỗi (string)
    age = 19           # Số nguyên (integer)
    height = 1.72      # Số thực (float)
    is_ready = True    # Đúng/Sai (boolean)
    ```
  </Step>
  
  <Step title="2. Điều kiện (if/else)">
    Ra quyết định dựa trên dữ liệu. Chú ý: Python dùng **thụt đầu dòng (indentation)** thay vì cặp ngoặc nhọn `{}`.
    ```python
    if age >= 18:
        print("Đủ tuổi bầu cử!")
    else:
        print("Chưa đủ tuổi.")
    ```
  </Step>
  
  <Step title="3. Danh sách (List) & Vòng lặp">
    Lưu nhiều món đồ vào một danh sách và duyệt qua chúng bằng vòng lặp `for`.
    ```python
    colors = ["đỏ", "xanh", "vàng"]
    for color in colors:
        print(f"Tôi thích màu {color}")
    ```
  </Step>
  
  <Step title="4. Hàm (Functions)">
    Đóng gói code để dùng lại nhiều lần, tránh việc copy-paste rác code.
    ```python
    def greet(user_name):
        return f"Chào {user_name}!"
    ```
  </Step>
</Stepper>

## Thực hành: Mini Project

Hãy thử viết một đoạn code nhỏ kiểm tra số chẵn lẻ:

```python
# 1. Yêu cầu nhập liệu
user_text = input("Nhập một số: ")

# 2. Chuyển chuỗi thành số nguyên
number = int(user_text)

# 3. Dùng phép chia lấy dư (%) để kiểm tra
if number % 2 == 0:
    print("Đây là số chẵn.")
else:
    print("Đây là số lẻ.")
```

<Callout type="warning" title="Bí kíp học nhanh nhất">
Đừng chỉ copy-paste. Hãy cố tình "phá" code: xóa dấu hai chấm, làm sai thụt dòng, gõ sai tên biến... rồi đọc kỹ thông báo lỗi. Lỗi chính là người thầy tốt nhất chỉ ra chỗ bạn làm sai!
</Callout>
