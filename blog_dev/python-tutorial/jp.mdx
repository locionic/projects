# まずは自分のPCで動くコードを書こう

動画のチュートリアルを見るのは楽しいですが、実際にコードを書かないと何も身につきません。環境構築から最初のプログラムまで、最速で駆け抜けましょう。

<Callout type="info" title="Step 0: Pythonのインストール">
**python.org** からダウンロードします。Windowsの場合、インストール時に **"Add Python to PATH"** にチェックを入れるのを絶対に忘れないでください！
</Callout>

---

## 押さえておくべき基本概念

<Stepper>
  <Step title="1. 変数とデータ型">
    Pythonは型の宣言が不要です。名前を付けて代入するだけ。
    ```python
    name = "Ava"       # 文字列 (str)
    age = 19           # 整数 (int)
    height = 1.72      # 小数 (float)
    is_ready = True    # 真偽値 (bool)
    ```
  </Step>
  
  <Step title="2. 条件分岐 (if/else)">
    条件によって処理を変えます。Pythonは `{}` ではなく **インデント（字下げ）** でブロックを表現します。
    ```python
    if age >= 18:
        print("投票できます！")
    else:
        print("まだ投票できません。")
    ```
  </Step>
  
  <Step title="3. リストとループ">
    複数のデータをリストにまとめ、`for` ループで処理します。
    ```python
    colors = ["red", "green", "blue"]
    for color in colors:
        print(f"I love {color}")
    ```
  </Step>
  
  <Step title="4. 関数">
    同じコードを何度も書かないように、処理を関数にまとめます。
    ```python
    def greet(user_name):
        return f"Hi, {user_name}!"
    ```
  </Step>
</Stepper>

## ミニプロジェクトを作ってみよう

偶数か奇数かを判定するプログラムを作ります。

```python
# 1. ユーザーに入力を求める
user_text = input("数字を入力してください: ")

# 2. 文字列を整数に変換する
number = int(user_text)

# 3. 割り算の余り (%) を使って判定する
if number % 2 == 0:
    print("その数字は偶数です。")
else:
    print("その数字は奇数です。")
```

<Callout type="warning" title="成長するためのコツ">
コピペで終わらせないでください。わざとコードを壊して（コロンを消す、インデントをずらす）、どんなエラーが出るか観察しましょう。エラーは「どこが分からないか」を教えてくれる先生です。
</Callout>
