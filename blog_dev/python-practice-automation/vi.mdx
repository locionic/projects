# Tự Động Hóa Các Tác Vụ "Click Khó Chịu" Bằng Python

Nếu bạn phải lặp lại một chuỗi click chuột hai lần, hãy nghĩ ngay đến tự động hóa. Mục tiêu không phải là xây một cỗ máy khổng lồ, mà là để tống khứ sự nhàm chán!

<Callout type="idea" title="Nguyên Tắc Vàng">
Bắt đầu cực nhỏ. Viết một đoạn script xử lý đúng một việc, kiểm tra cẩn thận, rồi mới để nó chạy ngầm.
</Callout>

---

## Quy Trình Tự Động Hóa Thực Dụng

<Stepper>
  <Step title="1. Chọn mục tiêu">
    Tìm tác vụ bạn ghét nhất: đổi tên file hàng loạt, dọn dẹp thư mục Downloads, xuất báo cáo CSV, hay gửi email báo cáo mỗi sáng.
  </Step>
  
  <Step title="2. Tận dụng Standard Library">
    Đừng vội cài thêm thư viện. `pathlib`, `shutil`, `csv`, và `datetime` là đủ cho 90% nhu cầu.
    ```python
    from pathlib import Path
    for p in Path("downloads").glob("*.csv"):
        print(f"Sẽ đổi tên file: {p.name}") # Luôn in ra test thử (dry run)!
    ```
  </Step>
  
  <Step title="3. Bảo vệ dữ liệu gốc">
    Không bao giờ ghi đè lên file gốc. Hãy đọc, xử lý (bằng `pandas` nếu phức tạp), và lưu kết quả ra một file hoàn toàn mới.
  </Step>

  <Step title="4. Lên lịch & Ghi log">
    Dùng **cron** (Linux/macOS) hoặc **Task Scheduler** (Windows). Đừng quên import `logging` để ghi lại lịch sử chạy, phòng khi lỗi xảy ra lúc nửa đêm.
  </Step>
</Stepper>

## Những Cái Bẫy Cần Tránh

<Callout type="warning" title="Đừng quá tin tưởng vào Input">
Tên file có thể sai format. API có thể bị timeout. Hãy luôn kiểm tra trước khi xử lý (validation) và dùng try/catch để script không bị "chết" âm thầm.
</Callout>

<Callout type="info" title="Bài Tập Bắt Đầu">
Hãy viết một script theo dõi thư mục Downloads và tự động chuyển các file `.jpg` sang thư mục Ảnh, `.pdf` sang Tài liệu. Đảm bảo bạn sẽ thích thành quả này!
</Callout>
