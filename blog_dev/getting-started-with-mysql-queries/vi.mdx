Khi mới bắt đầu làm việc với cơ sở dữ liệu, **MySQL** thường là hệ thống đầu tiên bạn tiếp cận. Nó phổ biến, mã nguồn mở, và cực kỳ mạnh mẽ cho mọi quy mô dự án.

Thay vì đọc các tài liệu dài dòng, đây là hướng dẫn ngắn gọn với 20% lệnh bạn sẽ dùng trong 80% thời gian!

<Callout type="info" title="Mẹo Cài Đặt">
  Nếu bạn chưa cài MySQL, cách nhanh nhất là dùng Docker: 
  `docker run --name mysql-dev -e MYSQL_ROOT_PASSWORD=secret -d mysql`
</Callout>

## 1. Kết Nối & Tạo Cơ Sở Dữ Liệu

<Stepper>
  <Step title="Truy cập qua Terminal">
    Để kết nối, sử dụng tài khoản của bạn:
    ```bash
    mysql -u root -p
    ```
  </Step>
  <Step title="Tạo mới & Chọn DB">
    ```sql
    CREATE DATABASE zoo_db;
    USE zoo_db;
    ```
  </Step>
</Stepper>

---

## 2. Tạo Bảng (Tables)

Giả sử chúng ta cần quản lý một sở thú với `habitat` (Môi trường) và `animal` (Động vật).

```sql
CREATE TABLE habitat (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(64)
);

CREATE TABLE animal (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(64),
  habitat_id INT,
  FOREIGN KEY (habitat_id) REFERENCES habitat(id)
);
```

<Callout type="idea" title="Sức mạnh của Khoá Ngoại (Foreign Key)">
  Khoá ngoại giúp bảo đảm tính toàn vẹn dữ liệu. Bạn không thể thêm một con vật vào một môi trường (`habitat_id`) không hề tồn tại!
</Callout>

---

## 3. Thêm, Sửa, Xóa (CRUD)

Thêm dữ liệu với `INSERT`, chỉnh sửa với `UPDATE`, và xóa với `DELETE`:

```sql
INSERT INTO habitat (name) VALUES ('River'), ('Forest');

UPDATE animal SET name = 'Quack' WHERE id = 2;

DELETE FROM animal WHERE id = 1;
```

<Callout type="danger" title="LUÔN SỬ DỤNG WHERE!">
  Nếu bạn chạy `UPDATE` hoặc `DELETE` mà không có `WHERE`, bạn sẽ sửa hoặc xóa **TOÀN BỘ** dữ liệu trong bảng. Hãy kiểm tra cẩn thận!
</Callout>

---

## 4. Truy vấn Dữ Liệu (Querying & GROUP BY)

Để thống kê tuổi trung bình của từng loài (loại bỏ con vật có id=3):

```sql
SELECT species, AVG(age) AS average_age
FROM animal
WHERE id != 3
GROUP BY species
HAVING AVG(age) > 3
ORDER BY AVG(age) DESC;
```

Để lấy tên động vật kèm tên môi trường của chúng (Kết hợp 2 bảng):

```sql
SELECT animal.name AS animal_name, habitat.name AS habitat_name
FROM animal
INNER JOIN habitat ON animal.habitat_id = habitat.id;
```

Hãy mở terminal lên và thực hành nhé. Đó là cách duy nhất để thực sự hiểu cách MySQL vận hành!
