# Empowering Agents with Seamless Payment Capabilities

Stripe's innovative Link wallet for agents introduces a new level of flexibility and control in payment processing. This cutting-edge feature provides agents with programmatic access to Link, unlocking a wide range of possibilities.

### Key Benefits of Link Wallet for Agents

* **Programmatic Access**: Agents can now access Link programmatically, streamlining their workflow and reducing manual intervention.
* **One-Time-Use Cards**: Agents can generate one-time-use cards, ensuring secure and efficient transactions.
* **Shared Payment Tokens (SPTs)**: Agents can create SPTs backed by existing cards and bank accounts, providing a seamless payment experience.

### Built on Stripe's Issuing for Agents

Link wallet for agents is built on top of Stripe's new Issuing for agents, a robust and scalable infrastructure designed to support the unique needs of agents. This foundation enables agents to manage their payment flows with ease, while maintaining the highest levels of security and compliance.

With Link wallet for agents, Stripe continues to push the boundaries of innovation in the fintech space, providing a powerful tool for agents to enhance their payment capabilities and drive business growth.
