# Cloudflare Acquires VoidZero: Empowering the Future of Web Development

Cloudflare is thrilled to announce the acquisition of VoidZero, the company behind the popular open-source toolchain Vite. This move demonstrates Cloudflare's commitment to supporting the JavaScript ecosystem and empowering developers to build a better internet.

## A Commitment to Open Source

As part of this acquisition, all VoidZero team members are joining Cloudflare, but the Vite toolchain will remain open source, vendor-agnostic, and community-driven. Cloudflare's mission is to help build a better internet, and a better internet is an open internet. The company believes that developers need choice, frameworks need a neutral foundation, and applications need to be portable.

## What This Means for Vite

With Cloudflare's support, Vite will continue to grow and evolve, with the same values that made it successful: speed, simplicity, and portability. The Vite roadmap will remain driven by the broader Vite team and community, and development will continue to be done in the open. Evan and the rest of the VoidZero team will continue to lead Vite, Vitest, Rolldown, Oxc, and Vite+.

## A $1 Million Commitment to the Vite Ecosystem

To demonstrate its commitment to the Vite ecosystem, Cloudflare is committing $1 million to a Vite ecosystem fund to support maintainers and contributors. This fund will be administered by the Vite core team and will help ensure the continued growth and success of the Vite toolchain.

## Vite as the Foundation

The Vite and Cloudflare teams have been collaborating for some time, and this acquisition is a natural next step. The Vite Environment API, which allows Vite to run server code in any runtime, has been a key area of collaboration. Cloudflare has built a Vite plugin on top of this API, which allows developers to run their server code in the same open-source runtime that powers Cloudflare Workers.

## The Future of Cloudflare's Tooling

Cloudflare is committed to moving its application tooling onto Vite, so it is built on top of the same workflows developers already know. The company recently shipped a technical preview of cf, a new unified CLI for the whole Cloudflare platform, which will be built on top of Vite.

## What Happens Next

In the short term, nothing changes for Vite users or the frameworks building on top of Vite. Vite, Vitest, Rolldown, Oxc, and Vite+ will continue to ship, and the VoidZero team will continue to contribute and lead them. Longer term, Cloudflare will start the work on moving the Cloudflare CLI toward an experience built directly on top of Vite, and Vite will get new, clean, provider-agnostic primitives for full-stack apps and agents.

## Welcome VoidZero

Vite, Vitest, Rolldown, Oxc, and Vite+ exist because of the hard work and dedication of the open-source community. Cloudflare is grateful to everyone who has contributed to these projects and is excited to welcome the VoidZero team to Cloudflare.

If you want to try Vite on Cloudflare today, run:

```bash
npm create vite@latest
npx wrangler deploy
```

This acquisition marks an exciting new chapter in the history of Vite and Cloudflare. Together, we will continue to empower developers to build a better internet.
