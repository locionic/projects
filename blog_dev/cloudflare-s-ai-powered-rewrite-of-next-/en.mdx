# Cloudflare's AI-Powered Rewrite of Next.js: A New Era for Commercial Open Source?

The recent announcement by Cloudflare that they have rewritten Next.js in just one week using AI has sent shockwaves through the developer community. This move has significant implications for the future of commercial open source and the role of AI in software development.

## The Next.js Ecosystem: A Recap

Next.js is the most popular full-stack React framework, used by around half of all React developers. The project is open source and primarily maintained by Vercel, which is also the preferred deployment target for Next.js applications. The reason for this is that Next.js applications are built using Vercel's proprietary Turbopack build tool, which produces a proprietary format that is optimized for Vercel's infrastructure.

## What Cloudflare Did with Next.js

Cloudflare replaced the Turbopack build dependency with Vite, a more standard build tool, to create a new package called vinext. This move allows Next.js applications to be easily deployed on Cloudflare's infrastructure, bypassing the proprietary build output of Vercel.

## AI Brings the Impossible Within Reach

Cloudflare claims that the rewrite was done by a single engineer using AI, at a cost of just $1,100 in tokens. This is a staggering achievement, considering that the original Next.js codebase has around 194,000 lines of code, while vinext has just 67,000 lines of code. The use of AI has made it possible to reduce the engineering time required for this task by up to 100x.

## "AI Slop" Still an Issue

However, there are concerns about the quality of vinext. Vercel's CEO, Guillermo Rauch, has pointed out that vinext is not production-ready and has security vulnerabilities. Cloudflare's announcement did not clearly state that vinext is experimental and not yet battle-tested, which has led to some confusion.

## New Attack Vector on Commercial Open Source?

The move by Cloudflare has raised questions about the defensibility of commercial open source. Vercel's strategy of building and maintaining Next.js, while optimizing Vercel to serve the specific build output, has been successful in generating business revenue. However, the use of AI to rewrite Next.js has brought the value of this strategy into question.

## Conclusion

The Cloudflare rewrite of Next.js using AI is a significant development that has far-reaching implications for the future of commercial open source. While there are concerns about the quality of vinext, the use of AI has made it possible to reduce engineering time and cost. As AI continues to evolve, we can expect to see more such developments that challenge the status quo in the software development industry.
