# The Pulse: Did Capacity Shortages Drive Anthropic's Changes to Claude Code?

In a surprising turn of events, Anthropic has been making changes to its Claude Code service that have left some developers feeling frustrated and confused. ## The Changes to Claude Code

In recent weeks, Anthropic has made several changes to Claude Code, including:

* Silently "nerfing" the service, reducing its capabilities without warning
* Banning corporate accounts without notice
* Implementing a weird growth experiment involving revoking and then restoring Claude Code access
* Removing Claude Code access from some paying customers, including those on the $20/month Pro plan

These changes have led some developers to speculate that Anthropic is intentionally trying to limit the use of Claude Code, possibly due to capacity constraints.

## Capacity Shortages

It appears that Anthropic has indeed been struggling with capacity issues for months. The company's CEO, Dario Amodei, recently revealed that Anthropic had seen 80x growth in revenue and usage over a short period, far exceeding its original plans for 10x growth. This rapid growth has likely put a strain on the company's resources, leading to capacity shortages.

The company has also entered into a deal with SpaceX / xAI, which will provide Anthropic with access to a large data center featuring over 220,000 NVIDIA GPUs. ## The AI Model Wars
