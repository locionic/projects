# Maximizing Claude's Free Plan

Most people assume Claude's free plan is just a crippled teaser designed to push you toward paying. That assumption is dead wrong!

The free plan in 2026 gives you the incredible **Claude Sonnet** model, a massive 200K context window, Projects, Artifacts, and — if you know how to set it up — local MCP tools via Claude Desktop. 

<Callout type="idea" title="The Secret Weapon: Projects">
  If you aren't using Projects, you're leaving value on the table. Upload your style guides, docs, or codebases once, set a system prompt, and Claude retains that context for every new chat inside the Project!
</Callout>

## Setting Up Local MCP (Model Context Protocol)

Claude Desktop is where local MCP becomes possible on the free plan, allowing Claude to interact with your local filesystem or terminal.

<Stepper>
  <Step title="Enable MCP">
    Open Claude Desktop → Settings → Developer → Enable MCP.
  </Step>
  <Step title="Configure Your Server">
    Edit your `claude_desktop_config.json` file to point to the `@modelcontextprotocol/server-filesystem`. Grant it access to specific folders on your hard drive.
  </Step>
  <Step title="Test It">
    Ask Claude: *"List the files in my Documents folder and summarize the text files."* It will actually read your local drive!
  </Step>
</Stepper>

## Built-In Connectors

Free users also get app connectors for Google Workspace, Notion, Slack, and Figma. Connect them and you can say: *"Summarize my last 3 emails about the project proposal."*

<Callout type="warning" title="Watch Your Quota">
  The free plan operates on a rolling 5-hour limit. To stretch it:
  - **Front-load context:** Write one detailed prompt instead of 5 short ones.
  - **Use off-peak hours:** You get much more leeway when servers are quiet.
  - **Keep chats focused:** Don't let a thread run for 40 messages if the topic changes; start a new one!
</Callout>

### When Should You Pay for Pro?

If you need **Claude Code** (the autonomous terminal agent), **Claude Opus** (the heavy-reasoning model), or remote API MCP connectors, the $20/month Pro tier is worth it. But for most users, mastering the free tier is more than enough!
