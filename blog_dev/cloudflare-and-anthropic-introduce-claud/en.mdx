# Cloudflare and Anthropic Introduce Claude Managed Agents on Cloudflare

Cloudflare and Anthropic have partnered to integrate Claude Managed Agents with Cloudflare Sandboxes, empowering developers to run agent-based workloads with greater control, security, and observability. This collaboration is the latest step in Cloudflare's mission to create the simplest, most secure, and most programmable cloud for agents.

## The Cloudflare Developer Platform Expansion

Over the past year, Cloudflare's Developer Platform has expanded to provide developers with the tools they need to run agents at scale. Key features include:

* **Sandboxes**: Full stateful Linux microVMs at scale, providing a secure environment for agent execution.
* **Agents SDK**: A simple and customizable agent framework for building and deploying agents.
* **Browser Run**: A fully programmable and observable browser for agents to interact with web applications.
* **Dynamic Workers**: Dynamic sandboxed code execution at massive scale, enabling agents to execute code in a secure and scalable manner.

## Introducing Claude Managed Agents on Cloudflare

The integration of Claude Managed Agents with Cloudflare Sandboxes provides a seamless way to run agent-based workloads on Cloudflare. With this integration, developers can:

* **Run agent loops on the Claude Platform**: Execute agent code on the Claude Platform while utilizing Cloudflare for code execution, secure connections, and custom tool calls.
* **Get started in minutes**: Use the default deployment template to quickly set up and deploy Claude Managed Agents on Cloudflare.

## Key Features of the Integration

The integration offers several key features, including:

* **Enhanced security**: Run all agent traffic through customizable proxies, enabling secure injection of credentials, prevention of data exfiltration, and better observability of agent interactions.
* **Sandbox control and observability**: Get detailed sandbox metrics and logs, SSH into running machines, and customize sandbox images.
* **Lightweight sandboxes**: Execute untrusted code in traditional microVMs or lightweight isolates, enabling massive scale, fast boot times, and minimized infrastructure spend.
* **Private service connectivity**: Connect agents to private internal services without exposing them to the internet.
* **Browser control and observability**: Get an audit trail of every agent's browser sessions, including session recording and human-in-the-loop flows.

## Securing Agent-Based Workloads

The integration provides several security features to protect agent-based workloads, including:

* **Customizable outbound proxies**: Define egress policies to expose only specific services to agent sandboxes, perform zero-trust injection of encrypted credentials, and access private services via Cloudflare Mesh.
* **Zero-trust authentication**: Use outbound Workers to handle egress and apply policies per tenant, per agent, or based on metadata.

## Extending Agent Capabilities with Cloudflare

The Cloudflare Developer Platform provides several tools to extend agent capabilities, including:

* **Browser Run**: Give agents fully programmable and observable browsers to interact with web applications.
* **Agent inboxes**: Provide agents with email addresses and the ability to send emails using the Cloudflare Email Service.
* **Custom tools**: Add customized tools to agents using the Cloudflare Workers platform.

## Conclusion

The integration of Claude Managed Agents with Cloudflare Sandboxes provides a powerful solution for running agent-based workloads with enhanced security, control, and observability. With the Cloudflare Developer Platform, developers can easily extend agent capabilities and build scalable, secure, and programmable cloud applications.
