# Claude Managed Agents on Cloudflare

Cloudflare and Anthropic have teamed up to integrate **Claude Managed Agents** with **Cloudflare Sandboxes**. This game-changing collaboration empowers developers to run agent-based AI workloads with unprecedented control, security, and observability.

<Callout type="idea" title="The Mission">
  Create the simplest, most secure, and most programmable cloud specifically tailored for autonomous agents.
</Callout>

## The Cloudflare Developer Platform

Cloudflare has expanded its tools to ensure agents can run securely at massive scale:

- **Sandboxes**: Full stateful Linux microVMs.
- **Browser Run**: A fully programmable browser for agents to navigate the web.
- **Dynamic Workers**: Sandboxed, scalable code execution.

## Getting Started with Managed Agents

Integrating Claude with Cloudflare Sandboxes provides a seamless deployment pipeline.

<Stepper>
  <Step title="Run Agent Loops">
    Execute agent code on the Claude Platform while utilizing Cloudflare for code execution, secure connections, and custom tool calls.
  </Step>
  <Step title="Secure Connectivity">
    Connect agents to private internal services without exposing those services to the public internet using Cloudflare Mesh.
  </Step>
  <Step title="Full Observability">
    Get detailed sandbox metrics, audit trails for browser sessions, and even SSH access into running agent machines!
  </Step>
</Stepper>

<Callout type="info" title="Zero-Trust Security">
  All agent traffic runs through customizable outbound proxies. This prevents data exfiltration and allows secure, zero-trust injection of encrypted credentials natively.
</Callout>

By leveraging the power of Anthropic's intelligence and Cloudflare's massive edge network, developers can build scalable, secure, and highly capable agentic applications in minutes.
