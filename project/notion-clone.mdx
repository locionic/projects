## Short Explanation

This project is a clone of the popular note-taking application Notion. It's built with Next.js as a learning exercise, following a tutorial on YouTube.

Some of the features that have been implemented include:

- 📋 Realtime database | [Convex](https://www.convex.dev/)

- 🔐 Authentication | [Clerk](https://clerk.com/)

- 💬 Image management | [Edgestore](https://edgestore.dev/)

- 👀 State management | [Zustand](https://github.com/pmndrs/zustand)

- 🖋 Rich text editor | [Blocknote](https://www.blocknotejs.org/)

- 🗑️ Trash can & soft delete

- 🔄 Recover deleted files

- 🌴 Tree structure for document

- 🔍 Search by using command + K

- 📝🌐 Publish/Unpublish note
