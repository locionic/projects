## Short Explanation

Some features will be added:

- Import/Export anime list from myanimelist
- User's setting

Some of the features that have been implemented include:

- 📋 Register/change password account

- 🔐 Authentication (token, recaptcha)

- 💬 Comment using disqus

- 👀 Watch/search/filter show, (random) episode

- 📝 Mark as planned/watching/on-hold/dropped/watched show

- 👀 Display/delete viewed show
